#  Airline Passenger Data Analysis (Mini Project)

##  Project Overview
This project is a beginner-level Exploratory Data Analysis (EDA) performed on an airline passenger dataset.  
The goal is to understand passenger details and flight status patterns using basic data analysis techniques.

This project was created as a **learning-focused mini project** while starting my journey as a Data Analyst.

##  Objectives
- Load and explore the airline passenger dataset
- Check for missing and duplicate values
- Understand data types and structure
- Analyze flight status distribution
- Perform simple statistical analysis
- Visualize key insights using charts

##  Dataset Information
- Source: Kaggle (Airline Passenger Dataset)
- Format: CSV / Excel
- Records include:
  - Passenger details (age, gender, nationality)
  - Airport and country information
  - Flight status (On Time, Delayed, Cancelled)

##  Data Cleaning Steps
- Checked dataset structure using `df.info()`
- Verified missing values using `df.isnull().sum()`
- Confirmed data types using `df.dtypes`
- Ensured dataset consistency before analysis
